## Decisions

- **TreeIndex over MPSC FIFO** — Stage 0 показала, что lockfree commit может финализировать WAL-entries вне порядка `commit_version`, FIFO недостаточно. `scc::TreeIndex` per ideology pillar #5.
- **Gap-reseed over duplicate-handling** — на дыру в окне один `wal.recover()` reseed + ретрай, вместо сложной dedup-логики. Steady state предполагается gap-free.
- **Best-effort offer, не блокирующий** — закрывает rust-intel §B14 unbounded-handoff: при overload `offer` молча drop'ает, gap-reseed подбирает позже. Коммит НИКОГДА не блокируется на drainer.
- **`pub mod drainer`** — расширил visibility (был `pub(crate)`) ради доступа из bench-файла. Минорное расширение поверхности, можно вернуть через `#[cfg(any(test, feature="bench"))]` шим follow-up'ом.
- **Pre-existing #163 вне scope кампании** — подтверждено git-stash-bisect'ом, что fail предшествует Op #2 (сломано в b2b1280). Per CLAUDE.md «не модифицирую код вне задачи».
