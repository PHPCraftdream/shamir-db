# OQL Roadmap — обзор

Четыре плана развития Object Query Language, выведенные из исследований
`docs/dev-artifacts/research/oql/01–04`. Вложенность батчей (research 01)
отдельного плана не требует — уже реализована; её улучшения растворены в
планах ниже.

| План | Тема | Сложность | Зависит от |
|---|---|---|---|
| [01](01-sequencing-explicitness.md) | Явность/симметрия зависимостей (`after`/`$query`) | низкая | — |
| [02](02-cond-value-evaluation.md) | Доэвалюировать `$cond`/`$expr` (условные значения) | низкая | — |
| [03](03-conditional-execution.md) | Условное исполнение op (`when`, if/switch) | высокая | 02 (жел. 01) |
| [04](04-loops-foreach.md) | Циклы (for-each по строкам, атомарный fan-out) | очень высокая | 02, 03 |

Рекомендуемый порядок: 01 → 02 → 03 → 04 (01 и 02 независимы, можно
параллельно). Каждый план включает фазы: движок, билдеры (Rust+TS),
юнит-тесты, e2e Rust, e2e TS, бенчмарки, доки. Планы 03 и 04 начинаются с
обязательного ADR.
