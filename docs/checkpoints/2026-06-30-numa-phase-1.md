## Decisions

- **NUMA Фаза 1 — только platform-independent skeleton, без
  LinuxTopology.** Каждая строка проверяема `cargo test` на Windows
  dev-хосте; CI mock-тесты уже едут в существующей `--workspace --lib`
  матрице ubuntu/windows/macos. Reject: писать LinuxTopology сразу
  «на глаз» — нельзя compile-check libc syscalls на Windows, риск
  красного CI.
- **NodeReplicated без cfg-gate'ов у потребителей.** Деградирует до
  одной реплики на single-socket → identical to bare ArcSwap, ноль
  overhead. Consumers used unconditionally. Reject: оставить старый
  `Arc<ArcSwap<T>>` для single-socket с `cfg`-switch — приносит
  cfg-шум в 9 ArcSwap-точек.
- **128-byte CachePadded alignment, не 64.** x86-64 adjacent-cache-line
  prefetcher + Apple Silicon 128-byte lines. Reject: 64 (формальный
  cache-line size x86-64) — false-sharing вернётся на ARM.
- **Phase P (dhat cross-check) deferred.** Q3-разбор показал что
  `registry.rs:89` peak=0 был ложной тревогой (код уже правильный),
  основное обоснование для cross-check снято. Reject: запустить dhat
  на «всякий случай» — нет evidence что нужно, ROI низкий.
- **`captrack-pgo measure` (T5) — single command UX вместо хелперов в
  shamir-db.** RAII CleanupGuard + cargo-metadata auto-detect.
  Reject: продолжать скриптовать wire→instrument→bench→merge→
  uninstrument→unwire руками — это и был UX-долг.
