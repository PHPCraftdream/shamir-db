## Decisions

- **Self-referential Cascade FK**: chose DDL-time rejection (`self_referential_cascade_not_supported`) over attempting genuine multi-level recursive support, because the existing `CascadePathGuard` cycle-detector is table-name-based and cannot distinguish a table cascading into itself from a real A→B→A cross-table cycle — a naive fix would false-positive-reject even the shallowest ordinary case (a manager with direct subordinates). RESTRICT and SetNull were enabled directly since neither recurses.
- **Aggregate `distinct` honoring**: chose to actually implement a generic `DistinctWrapper` for the funclib-dispatched aggregate path (clean, since it's already `Box<dyn Aggregator>`-based) rather than reject-everywhere, but chose reject-only for the closed fast-path (Sum/Avg/Count) since that hand-rolled state machine has no dedup-tracking capability without a heavier restructure — Min/Max allowed as a correct no-op either way.
