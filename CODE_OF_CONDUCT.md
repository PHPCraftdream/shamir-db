# Code of Conduct

## Our standard

We aim to keep this project welcoming, technically rigorous, and safe for everyone. Participants are expected to communicate with respect, assume good faith, give actionable feedback, and focus criticism on ideas and code rather than people.

Unacceptable behaviour includes harassment, discrimination, threats, sexualised conduct, doxxing, deliberate disruption, and publishing private information without permission.

## Enforcement

Report unacceptable behaviour privately to the repository maintainers through the contact options available in the GitHub repository. Do not use public issues for sensitive reports. Maintainers may remove content, close discussions, or restrict participation when necessary to protect the community.
